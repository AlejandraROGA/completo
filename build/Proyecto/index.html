PROYECTO

Higados de Hierro

¿Cómo surgió?

Esta idea surgio, por petición de un grupo de amigos, solian juntarse siempre en cada fiesta y anualmente hacen una posada para celebrar el fin de año y el comienzo de uno nuevo, pero sobre todo celebrando que aún están juntos todos. La idea principal del logotipo fue por el nombre que ellos mismos escogieron años atras HIGADOS DE HIERRO este haciendo homenaje a todas esas borracheras entre amigos.

Proceso de creación

Para conseguir el prototipo final, se tomaron varias ideas, incluyendo en ellas a la cebada, ingrediente principal para la cerveza, asi como dos tarros, los cuales representan el símbolo de amistad que existe entre estos amigos. El esquema de colores, fue seleccionado, siguiendo la referencia del color encontrado en las cervezas naturalmente (amarillo con toques cafesosos).

Conclusión

Al finalizar se entregaron tres ideas, las cuales conformaban todos los puntos escenciales para el logotipo, tipografía, esquema de color, referencias, etc. El cliente al escoger el prototipo final, tambien encargo camisas con el logotipo, estas para portarlas en la posada de fin de año, todo como un homenaje a una hermosa amista. Así fue como surgio la idea de HIGADOS DE HIERRO
